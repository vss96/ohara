## How to populate

```bash
# 1. Build a release binary so the numbers reflect the shipped profile.
cargo build --release --bin ohara

# 2. Fetch the pinned QuestDB fixture (shallow clone, ~5_875 commits).
./tests/perf/fetch_questdb.sh

# 3. Run the index with --profile and --no-progress so stdout is
#    just the summary line + the JSON. Pipe to jq if you want
#    pretty-printed output for the table below.
./target/release/ohara index --profile --no-progress \
    target/perf-fixtures/questdb \
    | tee target/perf-fixtures/questdb-baseline.json
```

Paste the resulting JSON into [§Raw output](#raw-output) and copy
each `*_ms` value into [§Per-phase wall-time](#per-phase-wall-time).
Compute `bytes_per_added_line = total_diff_bytes / total_added_lines`
and paste it into [§Hunk-text inflation](#hunk-text-inflation).
